Папка для своих картинок (рядом с index.html на хостинге).

logo.png
  Логотип в шапке. Рекомендуется квадрат PNG или JPG, от 128×128 px.
  Если файла нет — покажется эмодзи брокколи.

products/
  Имя файла = id товара из script.js + расширение .jpg (или поменяйте расширение в script.js):
    buckwheat_chicken.jpg
    rice_chicken.jpg
    beef_buckwheat.jpg
  Если файла нет — в карточке останется эмодзи.

banner-1.jpg, banner-2.jpg, banner-3.jpg (необязательно)
  Фон слайдера на главной. Если файла нет — используется только градиент.

После замены файлов загрузите папку web/ на GitHub Pages и обновите страницу в Telegram (потянуть вниз для перезагрузки Mini App).
